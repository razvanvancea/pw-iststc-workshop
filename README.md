# Scratch to Smart: Building a Scalable Playwright Framework with AI Agents and MCP Server

Welcome to the hands-on masterclass repository! This project will guide you through building a **scalable Playwright + TypeScript test automation framework** enhanced with **AI agents** (Planner, Generator, Healer) and MCP Server integration.

By the end of this session, you'll have a ready-to-extend project template and hands-on experience with AI-assisted test automation workflows.

---

## 🚀 Getting Started

### 1. Prerequisites

- Node.js (recommended v18+)
- npm
- Git
- Google Chrome browser
- Visual Studio Code

### 2. Clone the repository

```bash
git clone https://github.com/your-org/playwright-ai-masterclass.git
cd playwright-ai-masterclass

```

### 3. Install dependencies

```bash
npm install
```

### 4. Running tests

```bash
npm run test:check
```
